Python开发轻量级爬虫

发表于 2016-11-12   |   分类于 Blog   |     |   阅读次数

1.概述

对于轻量级爬虫：

  • 只抓取静态页面

而对于高级的爬虫，需要考虑的因素就较多，如：

  • 如何模拟登陆
  • 验证码识别
  • 分布式爬取网页
  • 数据存储
  • 网页更新

这些因素都具有重要的现实意义，但是对于初学者来讲，了解爬虫的基础架构显得更为必要，在实际应用场景中对其扩展可以爬取其他网页。

对于一个简单的爬虫架构，它包含了以下三大模块：
URL管理器： 用来管理将要抓取的URL和已经抓取过的URL
HTML下载器：给定一个URL，需要将其网页下载到本地后才能进行后续的分析。（这里使用Python自带的urllib2模块）
HTML解析器：得到了网页的内容以后，我们需要对其进行解析才能提取出我们想要的、我们感兴趣的内容。（这里使用Python的第三方插件BeautifulSoup）

2.爬虫架构：

爬虫调度端：
启动爬虫、停止爬虫或者监视爬虫的运行情况

而在爬虫程序中有三个模块：

URL管理器：对将要抓取的URL和已经抓取过的URL进行管理。我们可以取出一个将要爬取的URL传送给HTML下载器：，下载器会将URL指定的网页下载下来，存储成一个字符串，这个字符串会传送给HTML解析器：进行解析，一方面会解析出有价值的数据，另一方面每一个网页会有许多指向其他网页的URL，这些URL被解析出来以后可以补充进URL管理器。
URL管理器、HTML下载器和HTML下载器会形成一个循环，只要有相关联的URL就会一直运行下去。以上这些模块就组成了这个简单的爬虫架构。这个架构就可以把互联网上的相关所有网页下载下来。
